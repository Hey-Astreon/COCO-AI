01 / UNDETECTABILITY

Exempt from Screen Shares & Capture

CocoAI runs as a native Win32 window with setContentProtection(true). Because it operates at the operating system's window manager level, the overlay is completely invisible to Zoom, Google Meet, Teams, Discord, and local recording tools. You see the overlay on your monitor; your interviewer sees only your clean screen.

🔒

Zero Extension Footprint: No chrome extensions to detect or block.

🖥️

Dual-Device Ready: Stream transcription directly to your phone via local sync if screen sharing is restricted.

04 / INSTALLATION

Launch in 3 Minutes

CocoAI is open source. Setup is fast, developer-friendly, and completely private.

1

Clone the Repository

Clone the codebase and install dependencies on your local machine.

git clone https://github.com/Hey-Astreon/COCO-AI.git && cd COCO-AI && npm install
2

Configure Environment

Add your API keys to a .env file. Generous free tiers are available from all services.

CEREBRAS_API_KEY=your_key
DEEPGRAM_API_KEY=your_key
GEMINI_API_KEY=your_key
3

Run Application

Launch the Electron app overlay and begin your interview round.

npm start
